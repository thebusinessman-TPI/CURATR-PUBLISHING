# CURATR Canva Template Specifications

## 1. Overview

This document provides the official specifications for creating social media video templates in Canva for the CURATR platform. These guidelines are designed to ensure brand consistency, maximize visual impact, and optimize for performance across Instagram, TikTok, and YouTube Shorts.

**Key Objective:** Create a unified set of templates that can be easily adapted for each platform while maintaining a consistent brand identity for CURATR.

## 2. Universal Template Elements

These elements should be present in all video templates, regardless of the platform.

| Element | Description | Specifications |
| :--- | :--- | :--- |
| **Logo** | The official CURATR logo. | Use the approved logo file. Place in a consistent location (e.g., top-right corner). |
| **Brand Colors** | The official CURATR color palette. | Primary: `#667eea`, Secondary: `#764ba2`, Accent: `#f59e0b`. |
| **Typography** | The official CURATR font. | Use a clean, modern, and legible sans-serif font. Recommended: **Inter** or **Montserrat**. |
| **Call to Action (CTA)** | A clear and concise CTA. | Examples: "Shop the Collection," "Link in Bio," "See More at curatr.store." |

## 3. Platform-Specific Specifications

### 3.1. Instagram (Reels & Stories)

| Specification | Value | Notes |
| :--- | :--- | :--- |
| **Dimensions** | 1080 x 1920 pixels | Full-screen vertical format. |
| **Aspect Ratio** | 9:16 | Standard for Reels and Stories. |
| **Video Length** | 15-90 seconds | Reels can be up to 90 seconds, but shorter is often better. |
| **Safe Zones** | Keep critical elements within the center frame. | Avoid placing text or logos near the edges, as they may be covered by UI elements. |

### 3.2. TikTok

| Specification | Value | Notes |
| :--- | :--- | :--- |
| **Dimensions** | 1080 x 1920 pixels | Full-screen vertical format. |
| **Aspect Ratio** | 9:16 | Standard for TikTok videos. |
| **Video Length** | 15-60 seconds | While TikTok supports longer videos, the optimal length for engagement is typically under 60 seconds. |
| **Safe Zones** | Similar to Instagram, keep important content centered. | The bottom and right edges are where the caption, comments, and other UI elements appear. |

### 3.3. YouTube Shorts

| Specification | Value | Notes |
| :--- | :--- | :--- |
| **Dimensions** | 1080 x 1920 pixels | Full-screen vertical format. |
| **Aspect Ratio** | 9:16 | Standard for YouTube Shorts. |
| **Video Length** | Up to 60 seconds | Shorts are strictly 60 seconds or less. |
| **Safe Zones** | Keep key information in the center. | The bottom of the screen is used for the channel name, subscribe button, and other info. |

## 4. Content & Creative Best Practices

- **Hook within the first 3 seconds:** Grab the viewer's attention immediately.
- **Use trending audio:** Incorporate trending sounds and music to increase reach.
- **Fast-paced editing:** Keep the video dynamic and engaging with quick cuts and transitions.
- **On-screen text:** Use text overlays to highlight key messages and make the video accessible without sound.
- **High-quality video:** Ensure all footage is clear, well-lit, and visually appealing.
- **Looping content:** Create videos that can be watched on a loop to increase watch time.

## 5. Next Steps

1.  **Create a master template in Canva** with the universal elements.
2.  **Duplicate the master template** for each platform (Instagram, TikTok, YouTube Shorts).
3.  **Adjust the templates** to meet the specific requirements of each platform.
4.  **Share the templates** with the CURATR team for review and approval.
