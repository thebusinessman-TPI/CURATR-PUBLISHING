# Platform Content Specs

Quick reference for video content specifications across Instagram Reels, TikTok, and YouTube Shorts.

---

## Instagram Reels

### Video Specifications

| Spec | Requirement |
|------|-------------|
| **Aspect Ratio** | 9:16 (vertical) |
| **Resolution** | 1080 x 1920 px (recommended) |
| **Duration** | 15-90 seconds |
| **File Format** | MP4, MOV |
| **Max File Size** | 4 GB |
| **Frame Rate** | 30 fps (recommended) |

### Content Limits

| Element | Limit |
|---------|-------|
| **Caption Length** | 2,200 characters |
| **Hashtag Limit** | 30 max (3-5 recommended) |
| **Mentions** | No hard limit |

### Audio Rules

- Original audio, licensed music from Instagram library, or trending sounds
- Copyright-free music recommended for brand accounts
- Audio must not be muted or removed by Instagram
- Trending audio boosts discoverability
- Voiceovers can be added in-app

---

## TikTok

### Video Specifications

| Spec | Requirement |
|------|-------------|
| **Aspect Ratio** | 9:16 (vertical) |
| **Resolution** | 1080 x 1920 px (recommended) |
| **Duration** | 15-60 seconds (optimal), up to 10 minutes |
| **File Format** | MP4, MOV, WebM |
| **Max File Size** | 287 MB (mobile), 500 MB (web) |
| **Frame Rate** | 30 fps (recommended) |

### Content Limits

| Element | Limit |
|---------|-------|
| **Caption Length** | 2,200 characters |
| **Hashtag Best Practice** | 3-5 relevant hashtags |
| **Mentions** | No hard limit |

### Sound Requirements

- Original sounds, TikTok library, or trending sounds
- Videos with sound perform significantly better
- Trending sounds boost algorithm visibility
- Copyright music may be muted or cause video removal
- Text-to-speech available in-app

## YouTube Shorts

### Video Specifications

| Spec | Requirement |
|------|-------------|
| **Aspect Ratio** | 9:16 (vertical) or square |
| **Resolution** | 1080 x 1920 px (recommended) |
| **Duration** | Up to 60 seconds |
| **File Format** | MP4, MOV, AVI, WMV, MPEG, WebM |
| **Max File Size** | 256 GB (standard YouTube limit) |
| **Frame Rate** | 30 fps (recommended) |

### Content Limits

| Element | Limit |
|---------|-------|
| **Title Length** | 100 characters |
| **Description Length** | 5,000 characters |
| **Hashtag Usage** | Include #Shorts in title or description |
| **Tags** | 500 characters total |

### Audio Notes

- YouTube Audio Library for copyright-free music
- Original audio encouraged
- Some licensed music available through YouTube's agreements
- Audio may be muted in certain regions if copyrighted

## File Format Requirements

### Recommended Export Settings

| Setting | Value |
|---------|-------|
| **Container** | MP4 |
| **Video Codec** | H.264 |
| **Audio Codec** | AAC |
| **Video Bitrate** | 10-15 Mbps (1080p) |
| **Audio Bitrate** | 128-256 kbps |
| **Audio Sample Rate** | 44.1 kHz or 48 kHz |
| **Frame Rate** | 30 fps |
| **Resolution** | 1080 x 1920 px |

### Alternative Codecs

- H.265/HEVC (smaller file size, wider support growing)
- VP9 (YouTube optimized)

## Thumbnail Requirements

### Instagram Reels

- Auto-generated from video or custom cover image
- Recommended: 1080 x 1920 px (9:16)
- Appears in Reels tab and profile grid

### TikTok

- Select frame from video or upload custom cover
- Recommended: 1080 x 1920 px (9:16)
- Critical for profile grid appearance

### YouTube Shorts

- Auto-generated (cannot upload custom thumbnail for Shorts)
- Select best frame during upload
- Appears in Shorts shelf and channel page

## Best Practices Summary

1. **Export once, post everywhere** - Use 1080x1920, MP4, H.264 for all platforms
2. **Keep it short** - 15-30 seconds performs best across all platforms
3. **Add captions** - 85% of videos are watched without sound
4. **Use trending audio** - Especially on Instagram and TikTok
5. **Front-load content** - Hook viewers in first 3 seconds
6. **Include #Shorts** - Required for YouTube Shorts discovery
